# Runner 3D Asset Sources

- Runner characters and motion clips: RG Poly Cartoon City Massive Pack - Characters, CC0.
- City, suburban and industrial environment models: Kenney, CC0.

Source pages:

- https://rg-poly.itch.io/cartoon-city-massive-pack-characters
- https://kenney.nl/assets/city-kit-commercial
- https://kenney.nl/assets/city-kit-suburban
- https://kenney.nl/assets/city-kit-industrial

The environment models used by the game are merged and quantized into
`runner-city.glb` to reduce network requests and mobile transfer size.

The two local runner GLBs share the source pack skeleton and use the pack's
local motion library for reliable browser playback.
